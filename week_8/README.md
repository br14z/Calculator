# Food Delivery Receipt Generator

## Purpose of the Application

This application collects customer order information and generates a food delivery receipt.

It calculates:

- The subtotal
- A 5% service charge
- The delivery charge
- The final total

## Tech Stack

- Python
- GitHub
- Visual Studio Code

## Program Files

- `main.py` controls the application.
- `customer.py` collects customer information.
- `receipt.py` calculates and prints the receipt.

## How to Use

1. Open the `week_8` folder in Visual Studio Code.
2. Open the terminal.
3. Run the following command:

   python main.py

4. Enter the requested customer and order information.
5. The program will display the completed receipt.
