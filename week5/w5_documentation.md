# Week 5 Tutorial 5
## 1.1 Problem Statement
The cafe needs a simple Python program to calculate customer bills automatically.

## 1.2 Inputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 1.3 Outputs
- Customer receipt
- Quantity of each item
- Total bill in RM

## 1.4 Typical Process
1. Get customer name.
2. Get item quantities.
3. Multiply each quantity by its price.
4. Add all prices together.
5. Print the receipt.

## 1.5 Constraints
- Coffee price is RM 8.50
- Tea price is RM 6.00
- Sandwich price is RM 12.00
- Quantity should be a number.
- Quantity cannot be negative.

## 2. Decomposition
- Create a function to calculate total.
- Create a function to print receipt.
- Create main program to take inputs.
- Call the functions and show output.

## 3. Pseudocode
START
Ask for customer name
Ask for coffee quantity
Ask for tea quantity
Ask for sandwich quantity
Calculate total
Print receipt
END
